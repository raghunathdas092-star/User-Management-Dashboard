# 🚀 User Management Dashboard

A modern Full Stack User Management Dashboard built with Node.js, Express.js, MongoDB Atlas, and EJS.

---

## 📌 Overview

This application allows administrators to manage users securely through a web interface.

It demonstrates authentication, CRUD operations, session management, middleware implementation, and MongoDB integration following MVC architecture.

---

## ✨ Features

- User Registration
- Secure Login & Logout
- Session Authentication
- Create Users
- Update Users
- Delete Users
- Search & Filter Users
- Dynamic Dashboard
- MongoDB Atlas Integration
- Express Middleware
- Responsive UI

## ⚙ Installation

Clone the repository

git clone https://github.com/yourusername/User-Management-Dashboard.git

Install dependencies

npm install

Create a .env file

MONGO_URI=your_connection_string
PORT=5000
SESSION_SECRET=your_secret_key

Start server

npm start

Open browser

http://localhost:5000
